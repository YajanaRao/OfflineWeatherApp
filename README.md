## Setup

### Install the dependencies

```shell
# yarn
yarn install

# npm
npm install
```

### Install pods

```
pod install
```

### Run the metro bundler

```
yarn start
```

### Run the application on the device

```shell
# android
yarn android

# ios
yarn ios

# web
yarn web
```
